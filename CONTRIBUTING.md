## Contributing

All contributions are welcome, but please provide your reasoning for a proposed change or feature. Contributions should
align with the [goal](#goal) of the project.

### Goal

The Maven pom.xml template is designed to make collaboration easier by enforcing strict code style rules and providing a
recommended developer workflow. This project should also facilitate adapting the pom.xml to fit the specific use case.
Removing, modifying or adding dependencies and configuration should be encouraged.
